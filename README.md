# Bastion (archive)

> This repository is the original Bastion monorepo. It is preserved as a
> historical archive and is no longer where active development happens.

## The project has moved

Bastion was split into two repositories:

- **[bastion-core](https://github.com/thewaifucorp/bastion-core)** — the
  reusable Rust substrate: a workspace of crates (runtime, types, memory,
  cognition, personas, mesh, MCP, providers, agent-runtime, extension
  protocol). A host and mechanism, not a product.
- **[bastion-agent](https://github.com/thewaifucorp/bastion-agent)** — the
  personal agent product built on top of Core: the daemon/app, concrete
  channels, configuration, installer, mobile companion, skills and packs.

All active code, issues and releases now live in those two repositories.
This repository is not maintained going forward.

## Why it's kept around

This repo holds the complete development history of Bastion prior to the
split — every commit that shaped the runtime, the agent, and the decisions
along the way. It remains here, unaltered, as a historical record rather
than being deleted or rewritten.
